# ML_Music
```
Машинное обучение с аудиофайлами

Основные возможности:

Анализ аудио:
Определение типа звука (речь, музыка, шум)
Распознавание эмоций в голосе
Извлечение аудио-признаков (MFCC, спектральные характеристики, темп)

Визуализация:
Волновая форма
Спектрограмма
MFCC (Mel-кепстральные коэффициенты)
 
Обработка звука:
Шумоподавление (фильтрация высоких частот)
Эквалайзер (регулировка низких/средних/высоких частот)
```
